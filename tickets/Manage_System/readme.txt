# מערכת ניהול קריאות ומשימות

מערכת זו מאפשרת ניהול קריאות ומשימות עם התכונות הבאות:
- רישום למערכת עם חשבון Google
- ניהול קריאות ומשימות אישיות וכלליות
- התראות למייל על קריאות חדשות
- אפשרות לענות על קריאות, להעביר קריאות ולעדכן סטטוס

## התקנה

1. התקן את הדרישות:
   ```
   pip install -r requirements.txt
   ```
2. הגדר את משתני הסביבה (ראה `.env.example`)
3. בצע הגירות:
   ```
   python manage.py migrate
   ```
4. הפעל את השרת:
   ```
   python manage.py runserver
   ```
